Accessing PODD

Overview

There are public users and registered users in the PODD system. Registered users are standard users to the system. They have more privileges to the public users.

A public user can become a registered user by

  • Asking the system administrator to set up an account for them;
  • Registering themselves in PODD;
  • Using their Institutional Login via the Australian Access Federation.

Once they have an active account with PODD, they will be able to login or logout of the PODD system.

User registration and Login/Logout are discussed in the following sections in details

  • Self registration;
  • PODD login;
  • Institutional login;
  • Logout.
