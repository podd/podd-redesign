Filling Project Details


Project Details

The first section on the Create Project Object page is the project details section. Figure 10.1.1 is the screen shot of project details section, it includes the following fields (* denotes required field):

  • Title* : free form text
  • Description* : free form text, up to 1024 characters
  • Duration: expected duration, free form text
  • Start Date* : format YYYY-MM-DD, defaults to the current date
  • Lead Institution * : free form text field, however when a user types in an institution the field provides auto-completion capability, based on a list of institutions generated from the user objects.
  • Notes: free form text
  • URL:

 Figure 10.1.1 – Screen shot of the Project Details section on the Create Project Object page

The project status and publication status fields are not displayed in the Create Project screen, upon project object instantiation the default value for project status is Active and publication status is Not Published.

The users needs to fill in all the required fields. The Project Start Date is auto-filled with current date, but can be change to a different date.

No validation is performed until the Submit button at the end of the page is clicked.
