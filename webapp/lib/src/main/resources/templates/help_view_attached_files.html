Viewing Attached Files


File Attachments

A list of attached files are displayed in the File Attachments section. The file name, size, type, Format and Description are listed (see Figure 10.3.1.1).

A check box is displayed next to each file. By clicking the check box and Download Selected button, the user can download the files. If more than one file are selected, the downloaded files will be saved in a Zip file.

By clicking the Download All button in this section, the user can download all files. 

By clicking the File Name link, the user can view the matadata of the attached file, then edit or delete the file (see managing data items for details).
