Website Navigation


This section describes the basic look and feel of PODD interface. Please note that because the website is dynamic rather than static, the screen shots provided here may not be completely up to date, i.e. menus and content will change.

Navigation menu bar

This menu exists for all users, but varies depending on the userï¿½s repository role. For example: a public anonymous user may only see the tabs: Projects; Browse; and Search. A registered user may see: Projects; Browse; Search; and Clipboard. An administrator may see: Projects; Browse; Search; Clipboard; and Admin.

The Navigation Menu is currently left justified and has a distinct background and border. The development of the Navigation menu is also dynamic and more menu items may need to be added.

When the user has navigated to a menu selected function (e.g. Listing Projects via the Projects menu item) then the menu item will be highlighted (e.g. bolded and different colour).

Current Function Title

Displays the title of the current function the user has navigated to.

Main Interaction Pane

Primary interaction with PODD functions is through a single pane called the Main Interaction Pane. Some screens may have auxiliary panes as well. All screens will at least have the Main Interaction Pane.

Note that there will be the need for auxiliary panels that will act as menus when in particular functions. An example is for the Admin function. An auxiliary panel will be displayed on the LHS (user perspective) which will list admin functions. The Main Interaction Pane will display contents of the chosen admin function. Therefore the Main Interaction Pane must take up the space available after auxiliary panels are determined.

Search Box and Link

The search box and button are for basic searches. The Advanced Search link next to the Search button will allow more comprehensive searches via the Advanced Search screen. These items are visible on screen regardless of the current tab or user's access role.

User Functions Menu bar

User functions are listed in this menu bar, such as Settings, Login/Logout, Help, etc. This bar also displays the user’s full name. The User Functions Menu Bar is currently right justified, sitting above the Navigation Menu.
